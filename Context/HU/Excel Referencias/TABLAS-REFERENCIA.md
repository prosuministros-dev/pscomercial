## Notas de Uso

- **Formas de Pago**: Se utiliza para definir las condiciones de pago del cliente. Por defecto todos los clientes nuevos se crean como "ANTICIPADO".
- **Moneda**: Se utiliza para definir la moneda del costo de los productos en las cotizaciones.
- **Vía de Contacto**: Se utiliza para identificar el origen o canal por el cual llegó el lead.
- **Vertical**: Se utiliza para categorizar los productos. Las subverticales están pendientes de definición.
